# Formula Fit

Prototipo D2C de e-commerce hiper-personalizado para stacks de suplementos deportivos. SPA simulada en un solo `index.html` con HTML5 + CSS3 + JavaScript Vanilla (ES6). Sin dependencias, sin backend, sin base de datos.

## Cómo ejecutar

Abrir `index.html` en cualquier navegador moderno. No requiere servidor, build tools ni instalación de paquetes.

```bash
# Opción rápida desde terminal (Linux/macOS)
xdg-open index.html    # Linux
open index.html        # macOS
```

## Modo Simulación

Toda la aplicación opera en **Modo Simulación**: no se realizan transacciones reales, no se envían datos a ningún servidor y no se persiste información más allá de la sesión del navegador. El badge "Modo Simulación" permanece visible en todo momento como recordatorio.
